# Calculator_Project

**Простой Калькулятор** — учебный десктоп-проект на Python (Tkinter).  
Проект служит демонстрацией базовой командной работы: менеджер, разработчик, тестировщик.

---

## Коротко / Qisqacha
- Название: Calculator_Project  
- Описание: Простой калькулятор с GUI (кнопки 0–9, ., +, -, *, /, =, C, ←).  
- Рекомендация Python: 3.11 (работает и на других версиях, но 3.11 — гарантированно стабильна).

---

## Функции / Xususiyatlar
- Сложение, вычитание, умножение, деление
- Обработка деления на ноль (показывается "Error")
- Очистка (C) и удаление последнего символа (←)
- Поддержка ввода с клавиатуры

---

## Требования / Talablar
- Python 3.11+
- Библиотека Tkinter (обычно включена в стандартную поставку Python)
